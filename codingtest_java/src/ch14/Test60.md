### [문제 60] 두 행렬을 곱한 후 전치 행렬 만들기

정수값으로 이루어진 두 3x3 행렬 matrix1과 matrix2를 곱한 결과의 전치 행렬을 반환하는 solution 메소드를 구현하라

---

### 입출력의 예

| matrix1                             | matrix2                            | return                                         |
|-------------------------------------|------------------------------------|------------------------------------------------|
| [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] | [ [9, 8, 7], [6, 5, 4], [3, 2, 1] ] | [ [30, 84, 138], [24, 69, 114], [18, 54, 90] ] |
